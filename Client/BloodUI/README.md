## Requirements

- [BepInEx 6](https://builds.bepinex.dev/projects/bepinex_be) for V Rising
- [BloodCraft](https://thunderstore.io/c/v-rising/p/zfolmt/Bloodcraft/) server mod

## Installation

1. Install BepInEx 6 for V Rising
2. Copy `BloodUI.dll` to `BepInEx/plugins/`
3. Join a server running BloodCraft

## Configuration

After first run, edit `BepInEx/config/com.dinasor.bloodui.cfg`:

| Option | Default | Description |
|--------|---------|-------------|
| ExperienceBar | true | Show/hide XP bar |
| LegacyBar | true | Show/hide legacy bar |
| ExpertiseBar | true | Show/hide expertise bar |
| Familiars | true | Show/hide familiar bar |
| QuestTrackers | true | Show/hide quest tracker |
