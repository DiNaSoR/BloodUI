## Quick Start

1. Install [BepInEx 6](https://builds.bepinex.dev/projects/bepinex_be) for V Rising
2. Download `BloodUI.dll` from [Thunderstore](https://thunderstore.io/c/v-rising/p/DiNaSoR/BloodUI/)
3. Copy to `BepInEx/plugins/`
4. Join a server running BloodCraft
5. Enjoy the enhanced UI!

---

## Configuration

After first run, edit `BepInEx/config/com.dinasor.bloodui.cfg`:

| Option | Default | Description |
|--------|---------|-------------|
| ExperienceBar | true | Show/hide XP bar |
| LegacyBar | true | Show/hide blood legacy bar |
| ExpertiseBar | true | Show/hide weapon expertise bar |
| Familiars | true | Show/hide familiar bar |
| QuestTrackers | true | Show/hide quest tracker |
